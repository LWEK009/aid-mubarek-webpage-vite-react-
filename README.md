# 🌙 Eid Mubarak Romantic Website

A beautifully crafted, modern, and fully responsive **Eid Mubarak** webpage built with **React, Vite, and Tailwind CSS**. Designed specifically as a romantic and deeply personal greeting, this website combines **premium glassmorphism UI**, sophisticated **Arabic typography**, and stunning **micro-animations** to create an unforgettable user experience.

## ✨ Features

- **Premium Design Aesthetics**: Deep night-sky gradients, golden accents (`#d4a429`), and frosted glass (glassmorphism) cards to give a magical "wow" factor.
- **Dynamic Backgrounds**: Twinkling starfields, drifting golden particles, and floating ethereal lanterns (fanous).
- **Responsive Layout**: perfectly tailored to look beautiful on both mobile devices and large desktop screens.
- **Typography**: Uses classic and elegant Arabic fonts (`Amiri`, `Aref Ruqaa`, and `Tajawal`) for a distinguished, prestigious feel.
- **Classical Arabic Poetry**: Features a beautifully written classic Arabic poem (Fusha) with full Tashkeel (harakat), elevating the romantic and heartfelt message.
- **Interactive Audio**: Includes a subtle background music controller that gracefully auto-plays upon the user's first interaction.
- **Smooth Animations**: Tailored keyframe animations including `float-up`, `twinkle`, `heart-beat`, and a shimmering text effect that brings the website to life.

## 🛠️ Technology Stack

- **Framework**: [React 18](https://react.dev/) powered by [Vite](https://vitejs.dev/) for blazing-fast performance.
- **Styling**: [Tailwind CSS v3](https://tailwindcss.com/) customized with bespoke design tokens and custom keyframe animations.
- **Icons**: [Lucide React](https://lucide.dev/) for crisp, minimalist vector icons that align perfectly with the premium UI.

## 🚀 Quick Start

1. **Clone the repository**:
   ```bash
   git clone https://github.com/LWEK009/aid-mubarek-webpage-vite-react-.git
   cd aid-mubarek-webpage-vite-react-
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Add Background Music**:
   Place your chosen audio file (`eid-music.mp3`) inside the `public/` directory.

4. **Run the development server**:
   ```bash
   npm run dev
   ```

5. **Build for production**:
   ```bash
   npm run build
   ```

## 🎨 Project Structure

- `src/components/HeroSection.jsx`: The grand entrance with shimmering titles and names.
- `src/components/LoveMessages.jsx`: Heartfelt messages inside staggered glassmorphic cards.
- `src/components/PoemSection.jsx`: A classic Arabic poem using `Amiri` font and beautiful decorative ornaments.
- `src/components/PromiseSection.jsx`: A visually striking grid of romantic promises using Lucide icons.
- `src/components/FinalSection.jsx`: The closing section with a beating heart and closing prayers.
- `src/components/AudioControl.jsx`: The interactive audio widget.
- `src/components/*`: Background effect components (Stars, Moon, Lanterns, Particles).
